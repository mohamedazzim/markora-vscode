Markora uses the following direct dependencies. License text and exact transitive versions are retained by npm in
the installed package metadata and are locked in `package-lock.json`.

- React and React DOM; Tiptap/ProseMirror extensions — MIT
- Unified, Remark, MDAST, Marked, Turndown and GFM helpers — MIT
- Mermaid, KaTeX, DOMPurify, sanitize-html and Zod — MIT
- esbuild, TypeScript, Vitest, Testing Library and `@vscode/vsce` — MIT

The release process must regenerate a complete transitive notice bundle if Marketplace policy requires one. No
Typora code, CSS, icon, or proprietary asset is included.
