# Themes

The document theme is independent from the VS Code shell theme. **Classic White is the default**: it uses a pure
white document surface, dark neutral typography, blue links, subtle gray rules, pale code blocks, and a soft blue
selection inspired by the calm Typora writing surface shown in the product reference. `follow-vscode` remains
available as an explicit choice and uses VS Code editor variables. Markora also provides Paper, Academic, Sepia,
Graphite, Midnight, and High Contrast. Themes are implemented with document-scoped CSS variables and no external
theme CSS is downloaded.
